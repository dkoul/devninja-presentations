06 / 10
The agentic testing toolkit

Three Powerful Tools

Playwright MCP
Model Context Protocol server for Playwright. Let AI agents control browsers, take screenshots, and run tests through natural language.
Browser automation + MCP
WebMCP
Universal web interaction layer for AI agents. Navigate, click, fill forms, and extract data from any website programmatically.
Web interaction + Agents
AgentQL
Semantic selectors for the AI era. Query web elements using natural language instead of brittle CSS selectors or XPath.
Semantic querying + AI
Hands-on demos of each tool

Act V · Architecture

The layers

Five testing layers

1. Deterministic Testing
Traditional validation: API testing, schema validation, contract testing, authentication, infrastructure
2. Probabilistic Evaluation
Evaluation replaces assertions: relevance, groundedness, coherence, completeness, factual consistency
3. Adversarial Testing
Robustness against hostile input: jailbreaks, prompt injection, retrieval poisoning, obfuscation
4. Observability
Flight recorder: prompts, responses, reasoning, tool calls, retrieval sources, token usage, memory state
5. Continuous Evaluation
Post-deployment: production drift, user feedback, benchmark updates, adaptive risk recalibration
Five layers
